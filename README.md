Data Structure Headers
======================

This reposetory holds usefull C headers
with data structures and some algorithms.

Content Registry
----------------

- `types.h` - Java/Kotlin style primitive typedefs.
- `utils.h` - Utility macros.
- `array.h` - Size prefixed array macros.
- `buffer.h` & `buffer.c` - Java-like Buffer structure and functions.
